# Basic C Calculator

A simple console-based calculator written in C. This project is intended for beginners who are learning the basics of C programming, including user input, control flow, functions, and error handling.

## Features

- Addition, Subtraction, Multiplication, Division
- Input validation for numbers and menu choices
- Error handling for division by zero
- Menu-driven loop (perform multiple calculations until exit)

## Getting Started

### Prerequisites

- GCC or any standard C compiler

### Build & Run

1. Clone the repository:
   ```sh
   git clone https://github.com/ImadBengreine/Basic-C-calculator.git
   cd Basic-C-calculator
   ```
2. Compile:
   ```sh
   gcc main.c -o calculator
   ```
3. Run:
   ```sh
   ./calculator
   ```

## Usage

- Select the operation you want to perform from the menu.
- Enter two numbers when prompted.
- View the result.
- Repeat or exit as desired.
